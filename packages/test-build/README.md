### rollup
[rollup 官网](https://rollupjs.org/introduction/)
```text
- 适合纯js代码打包
- 不能打包ts,可以使用插件解决 @rollup/plugin-typescript
- 适合打包lib和组件
- 合并模块文件，或者拆分成单个文件
- 支持多种格式，esm,cjs等
- tree-shake特性，剔除未使用的代码，减少冗余
- 支持插件处理各种打包问题
```
